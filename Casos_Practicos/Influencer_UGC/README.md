# Caso Practico 2: Influencer Marketing + UGC

Material para sesion de live coding orientada a analisis de datos de creatividad digital.

## Archivos
- `datos/campanas_influencer_ugc.csv`
- `live_coding_caso_influencer_ugc_student.ipynb`
- `live_coding_caso_influencer_ugc_instructor.ipynb`
- `plantilla_reporte_caso_influencer_ugc.md`

## Habilidades que refuerza
- Limpieza y exploracion de datos
- Metricas de marketing digital (engagement rate, CTR link, CVR, CPA, ROAS, ROI)
- Filtros por segmento (tier, plataforma, tipo de contenido, pais)
- Estadistica descriptiva
- Outliers por IQR
- Reporte ejecutivo en Markdown
