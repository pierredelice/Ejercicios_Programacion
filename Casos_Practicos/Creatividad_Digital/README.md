# Caso Practico: Creatividad Digital

Este paquete contiene material para una sesion de **casos practicos + live coding**.

## Archivos
- `datos/campanas_creatividad_digital.csv`
- `plantilla_reporte_caso_creatividad.md`

## Objetivo de aprendizaje
- Analisis estadistico descriptivo sobre una BD realista
- Filtros por plataforma/formato/objetivo
- Deteccion de outliers (IQR)
- Graficas para storytelling de datos
- Redaccion de reporte tecnico en Markdown

## Flujo sugerido de clase (90-120 min)
1. Exploracion del dataset (15 min)
2. Metricas derivadas (15 min)
3. Filtros + preguntas de negocio (20 min)
4. Estadistica + outliers (20 min)
5. Graficas e interpretacion (20 min)
6. Reporte markdown (15 min)

## Entregables de estudiantes
- Notebook resuelto
- `reporte_caso_creatividad.md` completado
